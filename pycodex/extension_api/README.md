# pycodex.extension_api

Rust crate: `codex-extension-api`

Rust anchor: `codex/codex-rs/ext/extension-api`

This package mirrors the public crate interface exported from
`ext/extension-api/src/lib.rs`.  Extension registry/contributor/sink data
shapes are ported for compatibility; extension runtime behavior remains a
lightweight shim unless selected as its own module slice.
