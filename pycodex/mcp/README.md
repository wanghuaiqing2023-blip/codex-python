# pycodex.mcp

Rust crate: `codex-mcp`

Rust anchor: `codex/codex-rs/codex-mcp`

This package mirrors the public crate interface exported from
`codex-mcp/src/lib.rs`.  Configuration/status/auth/tool-provenance data shapes
and small helpers are ported for `codex-core` imports; external MCP server
runtime remains an explicit unported boundary.
