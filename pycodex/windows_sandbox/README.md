# pycodex.windows_sandbox

Rust crate: `codex-windows-sandbox`

Rust anchor: `codex/codex-rs/windows-sandbox-rs`

This package mirrors the public crate interface exported from
`windows-sandbox-rs/src/lib.rs`.  Small helper shapes are available for
`codex-core` imports; Windows token, ACL, elevated setup, and sandbox process
runtime remain explicit unported platform boundaries.
