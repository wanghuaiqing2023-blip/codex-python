# pycodex.rmcp_client

Rust crate: `codex-rmcp-client`

Rust anchor: `codex/codex-rs/rmcp-client`

This package mirrors the public crate interface exported from
`rmcp-client/src/lib.rs`.  OAuth status/discovery and client data shapes are
ported as dependency-facing interfaces; live Streamable HTTP/MCP transport is
not yet ported.
