# pycodex.exec_server

Rust crate: `codex-exec-server`

Rust anchor: `codex/codex-rs/exec-server`

Status: `complete` for the dependency-light Python port.

This package mirrors the public crate interface exported from
`exec-server/src/lib.rs`.  Protocol constants, request/response data shapes,
filesystem helpers, local process handling, remote registry helpers, JSON-RPC
transport projections, and server/session coordination slices are ported for
`codex-core` compatibility. Concrete network/TLS/runtime integrations remain
non-blocking operational checks rather than module or crate completion
blockers.
